# Kaiser Engineering Digital Dash GUI

For more information see the [wiki](https://wiki.kaiserengineering.io/en/gui)

## Install Deps

Requires Python3.7

```sh
pip install -r requirements.txt
```

## Running the DD

```sh
make run
```

## Testing

```bash
make test
make lint
```
